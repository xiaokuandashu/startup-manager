### 2.2 核心交互

| 功能 | 原理 |
|------|------|
| 选择电脑 | 通过 WebSocket 连接已登录的电脑列表 |
| 选择模型 | 远程读取目标电脑上的可用模型 |
| 控制开关 | 手机发指令 → WS → 电脑端执行 |
| 任务同步 | 手机创建/查看任务 ↔ 电脑同步显示 |
| 实时刷新 | 有时间差时手机可点击刷新 |
